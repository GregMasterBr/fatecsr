Checklist de SEO

Seguem diversos artigos que temos em nosso site para que vocês possam alinhar conhecimentos. Costumo dizer que um bom trabalho de SEO sempre começa arrumando a nossa casa. Para isso, precisamos entender alguns conceitos e trabalhar fortemente em nosso website.

Próximo

Pagerank

"Será que o PageRank influencia o meu site?"

O PageRank é basicamente a avaliação da relevância de determinada página. Essa relevância é divulgada
ao público numa escala de 0 a 10 por meio da
Toolbar do Google.

Veja o artigo
Próximo

Títulos

"Otimize um dos principais fatores de uma página."

Repita a title tag na h1 da página ou use algo semelhante. Assim, o usuário terá certeza de que chegou aonde queria e de que seu site oferece o
que ele precisa.

Veja o artigo
Próximo

Meta Description

"Aprenda a escrever uma boa Meta Description."

Meta Description Tags são tags que não aparecem no browser do usuário quando ele carrega uma página do site. Elas contêm informações voltadas principalmente para os motores de busca.

Veja o artigo
Próximo

Linkagem Interna

"Formas para conectar o conteúdo."

Deep links são links para páginas internas do site, e não para a inicial, nos quais existem níveis a serem cruzados até chegar à página desejada.

Veja o artigo
Próximo

URLs

"Aprenda como criar URLs interessantes para o Google e para o usuário."

Uma URL óbvia é uma ótima URL. Quando é possível saber qual é o conteúdo da página por meio da barra de endereços do navegador, antes mesmo de carregar esse conteúdo, temos uma URL perfeita.

Veja o artigo
Próximo

URLs Canônicas

"Para prevenir conteúdo duplicado no site."

Esse novo elemento visa ajudar os menos experientes a resolver problemas simples de conteúdo duplicado.

Veja o artigo
Próximo

Heading Tags

“Faça marcações importantes no conteúdo.”

As heading tags foram criadas para marcar pontos em destaque no conteúdo. Elas possuem um grau de importância no rankeamento e obedecem à mesma filosofia usada na leitura dos subtítulos de um texto.

Veja o artigo
Próximo

Imagens

"Valorize o conteúdo em forma de imagens.”

A otimização de imagens é um tema muito abordado
na área de SEO e abrange desde o uso de tags até o nome da imagem, o conteúdo no qual ela está
inserida e links.

Veja o artigo
Próximo

Breadcrumb

"Aprenda a conectar páginas para ajudar o Google e o usuário."

Breadcrumb Navigation são elementos que deixam as páginas mais bem qualificadas e ajudam o usuário a navegar melhor pelo site.

Veja o artigo
Próximo

Sitemap.xml

"Ajude o Google a descobrir mais páginas
do site."

Por meio desse tipo de Sitemap, é possível obter melhorias que ajudarão na indexação das páginas, pois as URLs serão acessadas mais facilmente pelo robô de busca.

Veja o artigo
Próximo

Página 404

"Como fazer o erro 404 trabalhar para você?"

Mantenha o layout padrão da página e mostre ao usuário que ele continua no site. Use links de navegação que permitam voltar ao conteúdo.

Veja o artigo
Próximo

Vídeos

"Otimize vídeos no WordPress."

Se você possui blog ou clientes com site no WordPress, deve conhecer o Video SEO para essa plataforma. Desenvolvido pelo Yoast, o plugin ajuda a incluir vídeos nas páginas e a aparecer com a thumbnail de vídeo nos resultados do Google.

Veja o artigo
Próximo

Redirecionamento

"Crie redirecionamentos
que ajudem o seu site perante o Google."

Para mudar o domínio sem perder nenhum link, visitante do site antigo ou PageRank, é necessário redirecionar todos os cliques do domínio antigo
para o novo.

Veja o artigo
Próximo

Robots.txt

"Crie regras e diga onde
as ferramentas de busca podem encontrar conteúdo."

Aprender como funciona o robots.txt é muito importante no processo de otimização de sites para evitar que algum conteúdo restrito fique disponível na web ou
que você sofra com conteúdo duplicado.
